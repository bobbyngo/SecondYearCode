~1.0 Exceptions~

Knowing how to handle exceptions is exceptionally important for writing robust and stable applications. Large systems are often riddled with areas where the state, control flow, or the results of an operation are uncertain. It is important to anticipate these problem areas and handle them accordingly. This section details some fundamentals on Exceptions in Java, and how to handle them.

~Task 1.5~

It is also possible to create custom exceptions. To do this, you should extend one of Exception or RuntimeException (more on this difference in Task 1.6).

Complete the MyCustomException class that is started for you. Then, change the customThrower method in Main.java so that it throws a MyCustomException. Make a try-catch block to handle it and print out the message in main().
