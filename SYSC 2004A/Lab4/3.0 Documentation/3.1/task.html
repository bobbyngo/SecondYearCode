~3.0 Documentation~

In Java, comments describing methods and classes are called document comments. They follow a standard structure so they can be converted into HTML for easy viewing. This is done by Javadoc, which is the generator that takes Java source code and converts it into viewable HTML.

Many IDEs (IntelliJ IDEA included) will generate this documentation automatically for you. You must simply adhere to the Javadoc format, which is to embed your documentation between /** */. I.e. above a method, type /**, press enter, and complete the document comment.
