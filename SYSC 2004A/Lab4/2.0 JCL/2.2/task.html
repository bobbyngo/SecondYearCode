The Java Class Library contains a plethora of built-in classes and interfaces. We have already seen a number of these such as String, Iterator, ArrayList, arrays, etc.

~Task 2.2~

We rarely use a single class or package in our programs. To import multiple classes from the same package we can use an *. For example, import java.io.*; will import all classes from the io package.

Fix the error in one.one.Main.java by using a single import line.
