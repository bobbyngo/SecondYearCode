The Java Class Library contains a plethora of built-in classes and interfaces. We have already seen a number of these such as String, Iterator, ArrayList, and arrays.

This section will serve as an introduction to some popular packages from the JCL.

~Task 2.1~

A very useful class is the Scanner class. Scanner belongs to the util package, which contains some other useful classes that we already know. Scanner can be used to read input from the console.

Read the docs for Scanner here: Oracle Docs: Scanner. As you may have noticed, many tasks refer you to Oracle Documentation if you are stuck. It is important to begin familiarizing yourself with the Java documentation, and learning how to extract the information you need from these extensive resources.

  1. In one.one.Main.java import the Scanner class so that it can be used. Create a new Scanner in the main() method.
    To read from console we need to provide the Scanner constructor with the System.in InputStream: new Scanner(System.in). And that is it! We do not need to worry about the details of how this works because it is all abstracted away for us!
  2. Go into the Scanner documentation and find a method that we can use to read a single line of text from console.
  3. Read a single line from console and store it in a variable.
  4. Echo the line back to console (print it!).

Note: to be able to type in the terminal you need to run one.one.Main.main() by clicking the small green arrow to the left of the main method signature.
