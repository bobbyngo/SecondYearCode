~Task 2.3~

java.lang.StringBuilder is useful when we need to perform a large number of String operations (e.g. many subsequent concatenations). Concatenating String objects together is computationally expensive, as under the hood a new String is being created for every concatenation.

Write code in one.one.Main.java to do the following:

  1. Continuously take input from console. The loop should quit when the user types in "quit".
  2. While taking input, each line should be appended to a StringBuilder object (as a new line).
  3. When the user enters quit, print out the StringBuilder (not including "quit").
Stuck? Consult the docs! -> Oracle Docs: StringBuilder

Note: to be able to type in the terminal you need to run one.one.Main.main() by clicking the small green arrow to the left of the main method signature.
