~Task 1.1~

As we already know, everything in Java is a class, i.e., everything is built from classes. We see these classes as lines of code in Main.java for example, but how does "the computer" know how to interpret this code and execute it? (Please note that the word interpret is used loosely here.)

The first step in the process is the compilation of .java files into bytecode. If you compile and run your Java programs from a command prompt, you would use a program called javac to first compile your .java file. javac is a Java compiler. If you then took a peek into your directory, you would find a .class file. This .class file contains the bytecode.
