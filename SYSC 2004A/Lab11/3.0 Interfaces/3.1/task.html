~3.0 Interfaces~

This section provides a brief introduction to interfaces. There is a lot to know when it comes to specific details on interfaces in Java. Therefore, this lab (and course) gives a high-level view -- a conceptual view of interfaces in object-oriented programming.

~Task 3.1~

An interface defines a contract for an implementing class to follow. When a class implements an interface, it is saying "I promise to do (implement) what this interface has declared derived classes should do". In this way we can enforce rules, and establish "frameworks" for inter-class communication.
