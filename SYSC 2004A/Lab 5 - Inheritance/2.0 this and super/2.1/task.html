~2.0 this and super~

this and super are a couple of handy reserved words in Java. We have already seen this used fairly extensively, but have never discussed it. Conceptually they are simple to understand, and that is what we will be doing in this section.

~Task 2.1~

this is a reserved word used to access the current object reference. What is the current object reference? For example, if we call the method evolve() on a Virus object v1, within the context of that method call "this" will refer to the calling object v1. this is useful in situations where we do not have access to the current object's reference. The three cases in which we need this are:

  • When we need to disambiguate variables. Example: in the Virus constructor.
  • When we want to pass the current object reference to another method. Example: anObject.someMethod(this).
  • Constructor chaining. Example: consider the revised version of Virus.java with constructor chaining. Note: a call to this() must always be the first line in a constructor body.
